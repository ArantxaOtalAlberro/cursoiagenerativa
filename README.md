# curso-ia-generativa

Repositorio con ejercicios y laboratorios del curso "IA Generativa".

Estructura principal:
- `laboratorio 3/`, `laboratorio 4/`, `laboratorio 5/` — carpetas con ejercicios.

Uso rápido:
```bash
# Ejecutar un ejemplo de Python
python3 "laboratorio 5/my_project/python_module/example.py"
```

Buenas prácticas:
- No imprimir contraseñas ni usar `eval` en JavaScript.
- Mantener formato con `ruff` y revisar seguridad con `semgrep`.
